First published project - WebSnap

This was the very first bigger project we did in school. It is a native android application written in Java. The goal of the project was to make a application that would enable me to get a piece of a guide site for League of Legends in my phones home screen so I would always see the free champion rotation. Lol was all the craze back then(Still playing from time to time btw). I had a few apps in my phone that could technically do this, but they used a method to select HTML elements in the site and just fetch those every time the widget updated. I got to the conclusion that this method was a pain for the user and decided to find a way to get an actual image out of the site. I implemented a mask that the user would position in the site, a screenshot and crop functions and all the UI needed for this. The application is working and usable, but has a few crashing bugs and has to show the full page every time it updates the widget. Never got back to the project and it had to be shipped by the end if the course, so that's the state it was left in.

Finnish Defence forces - Sight adjusting game

This was my conscript time. I got selected to the ADL-team based in Tuusula and was to work on a first project on a suite of educational games meant to possibly replace some classes at the beginning of a new conscripts serving time and teach some basic skills in a new and more engaging way. The end result was a complete 3D game where the user can shoot and adjust the sights of the same weapon used in the real drill in a 3D enviroment and see the actual result when shooting again. This has not released yet, but is undergoing small changes in the hands of the new conscripts in the ADL-team and will be released by Finnish Defence forces in the near future.
